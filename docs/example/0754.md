## 打印 Hello World 的示例 Python 程序

这个示例 Python 程序打印了 hello world。下面的打印语句用 Python 打印 Hello World。

```
# Sample Python Program to Print Hello World

print("Hello World")
```

你好世界节目输出

```
Hello World
```
