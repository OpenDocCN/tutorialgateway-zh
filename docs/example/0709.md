# Go 程序：打印简单的数字模式

> 原文：<https://www.tutorialgateway.org/go-program-to-print-a-simple-number-pattern/>

编写一个 Go 程序，打印一个简单的数字模式，用于循环。

```go
package main

import "fmt"

func main() {

	var i, j, row int

	fmt.Print("Enter Simple Number Pattern Rows = ")
	fmt.Scanln(&row)

	fmt.Println("**** Simple Number Pattern ****")

	for i = 1; i <= row; i++ {
		for j = 1; j <= i; j++ {
			fmt.Printf("%d ", j)
		}
		fmt.Println()
	}
}
```
