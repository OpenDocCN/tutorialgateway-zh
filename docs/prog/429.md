## 如何在 SQL Server 中交换列值？

请使用下面显示的频繁查询来交换两个列值。

```
UPDATE [dbo].[EmployeeDuplicates]
	SET [Education] = [Occupation],
	    [Occupation] = [Education]
GO
```

```
Messages
--------
(14 row(s) affected)
```
